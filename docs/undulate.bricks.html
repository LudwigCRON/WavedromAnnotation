undulate.bricks module¶

bricks.py declare the basic building block to generate a waveform

class undulate.bricks.BRICKS(value)¶

Bases: enum.Enum

BRICKS enumerate the different allowed block and symbol to describe a waveform

It defines the mapping between symbols and their character representation inside the input file
