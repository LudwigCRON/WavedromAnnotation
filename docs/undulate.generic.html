undulate.generic module¶

bricks.py declare the basic building block to generate a waveform

class undulate.generic.Brick(**kwargs)¶

Bases: object

define the brick as a composition of paths, arrows, and generic polygons to fill an area

Variables

  • width (float > 0) – by default is 40

  • height (float > 0) – by default is 20

  • slewing (float > 0) – by default 0

  • duty_cycle (float > 0) – between 0.0 and 1.0 by default 0.5

  • ignore_transition (bool) – by default False

  • is_first (bool) – first brick of a wavelance, by default False

  • last_y (float) – y coordinate of the previous brick to make the junction between the two

  • equation (str or float) – analogue value(s)

  • paths (list) – list of “svg” paths to be drawn

  • arrows (list) – list of arrows to be drawn

  • polygons (list) –

    list of polygons to be drawn

    Warning

    polygons are considered to be a list of (x, y) tuples

  • splines (list) –

    list of splines to be drawn

    Warning

    splines are considered to be a list of (type, x or dx, y or dy) tuples for more details please look at https://developer.mozilla.org/en-US/docs/Web/SVG/Tutorial/Paths
